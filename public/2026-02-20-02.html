谷歌浏览器免费解锁 Gemini 完全体？别人没告诉你的细节，都在这里（保姆级教程）

杆子 2026-02-19 16:35  教程 2368 字  5 分钟  编辑

谷歌最近终于干了件大事——把 Gemini 直接塞进了 Chrome 浏览器，实现了真正的 AI 浏览器。

这不仅仅是在侧边栏加个聊天框，而是把 AI 变成了浏览器的”操作系统”。现在，你打开 Chrome，右上角就能唤起 Gemini：总结网页、对比价格、自动填表、分析视频，甚至直接跨标签页处理信息，一气呵成。

但在国内，由于严格的地区检测，99% 的用户即便更新了浏览器也看不到这个图标。

很多教程只告诉你改个设置，结果重启后还是没反应。今天我写了这篇保姆级教程，教你如何使用上这个基础设施级的工具。我把每一个细节都补全，手把手教你在 Windows 和 Mac 上强行开启这个”AI 完全体”Chrome。

⚠️ 核心提示：开始前的心理建设

开启该功能属于”强行解锁”，需要将电脑系统地区修改为”美国”。

如果你因工作软件兼容性（如某些财务软件、税务软件）介意修改系统地区，那么目前可能无法使用。请在接受此前提后再继续。

一、为什么用

Gemini in Chrome，而不是 ChatGPT Atlas？ {#geminiinchromechatgptatlas}

我之前是 OpenAI 的 ChatGPT Atlas 重度用户，但实测下来，我有三个痛点实在忍不了：

  1. 太贵了：Atlas 免费版模型很弱，想用强的得开 Plus 会员，一个月 20 美金，一年就是 1700+ RMB。（在 Gemini 3 出来之前，我一直是会员）
  2. 效率低，限制多：Atlas 目前只能在一个标签页下”单打独斗”。不能在多个网页下调用，但这真的是刚需。

3. 模型受限：Chrome 里的 Gemini 可以直接调用 Gemini 3 Pro、Flash 甚至最新的 Thinking 模型。最重要的是，它支持 Multimodal（多模态）+ Multi-tab（多标签页）。

场景举例：你可以同时打开京东和淘宝的商品页，直接问浏览器：”这两个页面里的某某商品，谁更划算？”它能同时读取并对比，这才是真正的生产力。

二、准备条件

（缺一不可） {#-1}

  1. 浏览器版本：Chrome 131 及以上版本（在地址栏输入 chrome://settings/help 检查），建议直接下载最新版本。
  2. 网络环境：全程保持干净的美国节点（这是基础）。
  3. 系统、浏览器地区：系统和浏览器的地区均需设为美国。
  4. Gemini Pro 会员：必须要拥有一个 Gemini Pro 会员 的 Google 账号（这是解锁”自动浏览”和顶级模型的关键）。

三、详细开启教程

（Windows & Mac） {#windowsmac}

此过程涉及修改系统文件，请严格按照顺序操作。

第一步：彻底关闭

Chrome {#chrome}

由 90% 的失败源于此——很多人改了文件没生效，是因为 Chrome 只是关闭了窗口，后台进程还在运行，导致修改被覆盖。

  • Windows 用户：打开任务管理器（Ctrl+Shift+Esc），找到所有 Google Chrome 进程，右键点击 “结束任务”，确保一个不留。
  • Mac 用户：在 Dock 栏对着 Chrome 图标右键 -> 退出，或者打开”活动监视器”强制退出所有 Chrome 相关进程。

第二步：修改本地配置文件

找到 Chrome 的 Local State 文件，这是一个没有后缀名的 JSON 文件。

文件路径：

  • Windows: C:\Users\你的用户名\AppData\Local\Google\Chrome\User Data\Local State
  • Mac: ~/Library/Application Support/Google/Chrome/Local State

1. 修改 variations_country

  • 原代码地区可能是："cn" 或其他地区
  • 改为："us"
  • 注意：保留前面的数字版本号，只改国家代码。

2. 修改 variations_country

（第二处） {#2variations_country}

  • 原代码："variations_country":"cn"
  • 改为："variations_country":"us"

3. 修改/添加 is_glic_eligible

  • 搜索 is_glic_eligible
  • 如果找到了，把值改为 true（可能有好几处，都修改一下）
  • 如果找不到：请在文件靠前的位置（例如第一个大括号 { 之后）手动添加一行："is_glic_eligible": true,

全部修改完后，保存文件并关闭。

第三步：添加启动参数

（增加成功率的”保险”） {#-3}

很多教程忽略了这一步，导致功能时有时无。

Windows 用户

  1. 右键点击桌面的 Chrome 快捷方式 -> 属性
  2. 找到 “目标” 一栏，在原有路径的最后面，先打一个空格，然后粘贴以下代码：

--enable-features=OptimizationGuideModelDownloading,OptimizationHints
  1. 点击确定。以后必须通过这个快捷方式启动 Chrome。

Mac 用户

打开终端（Terminal），输入以下命令后回车（这是最稳妥的方式）：


open -a "Google Chrome" --args --enable-features=OptimizationGuideModelDownloading,OptimizationHints

第四步：系统与浏览器环境”伪装”

Windows 设置

打开”设置” -> “时间和语言” -> “区域”，将”国家或地区”改为 美国。

Mac 设置

打开”系统设置” -> “通用” -> “语言与地区”，将”地区”改为 美国。

Chrome 内部语言设置

（关键） {#chrome-1}

打开 Chrome 设置 (chrome://settings/languages)，将 English (United States) 添加并置顶，勾选”以这种语言显示 Google Chrome”。这一步非常重要，中文界面极大概率无法触发 Gemini。

第五步：开启

Flags 开关 {#flags}

重启浏览器后，在地址栏输入 chrome://flags，搜索 “glic”，将出现的以下两项从 Default 改为 Enabled：

  • #glic (Gemini Live in Chrome)
  • #glic-side-panel

最后点击右下角的 Relaunch 再次重启。

四、成功的标志与功能演示

如果一切顺利，重启后你会发现：

  1. 地址栏右侧（或头像旁）出现了一个带星号的 Gemini 图标
  2. 点击图标，侧边栏弹出，即可进行对话

你可以立刻尝试：

  • 网页摘要：打开一篇长新闻，点开 Gemini，它会自动生成 Summary
  • 多方比价：同时打开两个电商产品页，问它：”Compare the price and features of these two tabs”（对比这两个标签页的价格和功能）
  • 视频分析：打开一个 YouTube 视频，直接问它视频里讲了什么，它能秒出总结
