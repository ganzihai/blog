免费大模型API，每天都能用！

杆子 2026-04-16 12:26  教程 1738 字  4 分钟  编辑

免费的东西大家都喜欢，尤其是大模型 API 这种按量计费的。本文整理了一批目前仍能免费使用的大模型 API，涵盖国内外的主流厂商，适合个人开发者、学生和小型项目使用。

1. Google Gemini API

免费额度

  • Gemini 2.0 Flash / Flash-Lite：15 RPM（每分钟请求数），150 万 Token/天
  • Gemini 2.0 Pro：2 RPM，5 万 Token/天

使用方式

  1. 访问 Google AI Studio
  2. 点击 “Get API Key”，创建 API Key
  3. 在代码中使用，示例：
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")
model = genai.GenerativeModel('gemini-2.0-flash')
response = model.generate_content("你好，请介绍一下自己")
print(response.text)

注意事项

  • 免费版本的数据可能会被 Google 用于改进模型
  • 有速率限制，不适合高并发场景
  • 输出内容可能包含 Gemini 的水印

2. OpenAI API

免费额度

  • 新注册用户：赠送 $5 额度（有效期 3 个月）
  • gpt-4o-mini：$0.15/百万输入 Token，$0.6/百万输出 Token

使用方式

  1. 注册 OpenAI 账号
  2. 在 API Keys 页面创建新的 Key
  3. 使用示例：
from openai import OpenAI

client = OpenAI(api_key="YOUR_API_KEY")
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)

注意事项

  • 免费额度用完后需要充值
  • 中国大陆访问可能需要代理
  • API 调用有速率限制

3. 百度文心一言 API

免费额度

  • ERNIE-Bot：每日 10 万 Token 免费
  • ERNIE-Speed：每日 50 万 Token 免费

使用方式

  1. 注册 百度智能云账号
  2. 开通千帆大模型平台
  3. 创建应用获取 API Key 和 Secret Key
import requests

def get_access_token(api_key, secret_key):
    url = f"https://aip.baidubce.com/oauth/2.0/token?grant_type=client_credentials&client_id={api_key}&client_secret={secret_key}"
    response = requests.post(url)
    return response.json().get("access_token")

def chat(access_token, prompt):
    url = f"https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/eb-instant?access_token={access_token}"
    payload = {
        "messages": [{"role": "user", "content": prompt}]
    }
    response = requests.post(url, json=payload)
    return response.json().get("result")

注意事项

  • 需要实名认证
  • 有调用频率限制
  • 部分高级模型需要付费

4. 阿里通义千问 API

免费额度

  • qwen-turbo：每日 100 万 Token 免费
  • qwen-plus：新用户赠送额度

使用方式

  1. 注册 阿里云账号
  2. 开通百炼平台
  3. 创建 API Key
from dashscope import Generation

def chat(prompt):
    response = Generation.call(
        model="qwen-turbo",
        prompt=prompt,
        api_key="YOUR_API_KEY"
    )
    return response.output.text

注意事项

  • 需要实名认证
  • 有调用频率和并发限制
  • 部分模型有每日免费额度

5. 智谱 AI (GLM)

免费额度

  • glm-4-flash：每日 100 万 Token 免费
  • glm-4-air：新用户赠送额度

使用方式

  1. 注册 智谱 AI 账号
  2. 创建 API Key
  3. 使用示例：
from zhipuai import ZhipuAI

client = ZhipuAI(api_key="YOUR_API_KEY")
response = client.chat.completions.create(
    model="glm-4-flash",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)

注意事项

  • 需要实名认证
  • 有调用频率限制
  • 免费模型输出质量相对一般

6. Moonshot (Kimi)

免费额度

  • moonshot-v1-8k：新用户赠送 15 元额度
  • moonshot-v1-32k：支持更长上下文

使用方式

  1. 注册 Moonshot 账号
  2. 获取 API Key
  3. 使用示例：
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.moonshot.cn/v1"
)
response = client.chat.completions.create(
    model="moonshot-v1-8k",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)

注意事项

  • 兼容 OpenAI SDK
  • 免费额度用完需充值
  • 长上下文模型价格较高

7. MiniMax

免费额度

  • abab6.5：新用户赠送 1000 万 Token
  • abab5.5：每日有一定免费额度

使用方式

  1. 注册 MiniMax 账号
  2. 获取 API Key
  3. 使用示例：
import requests

def chat(prompt):
    url = "https://api.minimax.chat/v1/text/chatcompletion_pro"
    headers = {"Authorization": "Bearer YOUR_API_KEY"}
    payload = {
        "model": "abab6.5",
        "messages": [{"role": "user", "content": prompt}]
    }
    response = requests.post(url, headers=headers, json=payload)
    return response.json().get("reply")

注意事项

  • 需要实名认证
  • 免费额度有限
  • 适合中文场景

9. 硅基流动 (SiliconFlow)

免费额度

  • 聚合多个开源模型，每日赠送一定额度
  • 支持 Llama、Qwen、GLM 等模型

使用方式

  1. 注册 SiliconFlow 账号
  2. 获取 API Key
  3. 使用示例：
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.siliconflow.cn/v1"
)
response = client.chat.completions.create(
    model="Qwen/Qwen2.5-72B-Instruct",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)

注意事项

  • 兼容 OpenAI SDK
  • 模型选择丰富
  • 免费额度每日刷新

  • 适合代码生成和推理场景
