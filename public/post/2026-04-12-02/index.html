安装 Claude Code

在 Windows PowerShell 中执行以下命令进行安装：

curl -fsSL https://claude.ai/install.sh | bash

安装 CC Switch

项目地址： https://github.com/farion1231/cc-switch

下载软件： https://github.com/farion1231/cc-switch/releases/tag/v3.12.3

下载并安装 CC-Switch-v3.12.3-Windows.msi 文件即可。
