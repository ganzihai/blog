影音党狂喜！全能音视频播放器，多源聚合+纯本地，解锁极致体验

杆子 2026-03-19 19:43  教程 1073 字  3 分钟  编辑

原创：凹凸曼 | 数码凹凸曼

作为影音爱好者，你是否有过这样的烦恼：

  • 影音资源散落在不同服务器，切换查找繁琐；
  • 多设备播放体验不统一；
  • 担心隐私泄露不敢随意用播放器？

今天继续做仓库搬运工，给大家推荐一个在 GitHub 上有 1.5K Star 的音视频播放器，名字是 Cinetry。

项目介绍与获取

这个工具号称全能音视频播放器，适配 Android（Phone+TV）、Windows、macOS、Linux 等多平台，甚至 iPhone 也可以借助巨魔等工具进行安装。最重要的是完全免费，没有内置广告。

  • GitHub 地址： https://github.com/gstory0404/Cinetry
  • 官网地址： https://cinetry.gstory.cn/

核心亮点

1. 强大的多源聚合与服务器管理

Cinetry 最核心的亮点，就是强大的多源聚合与服务器管理能力。它原生支持 Jellyfin、Emby、IPTV、WebDav 等数十种主流媒体服务器，电影、电视剧、音乐、图片等各类媒体库都能一键同步。

你无需在多个客户端之间来回切换，打开就能快速访问所有存储内容，彻底告别资源分散的烦恼。同时支持自定义元数据规则，让不同平台的资源格式保持统一，整理资源再也不用费心思。

更贴心的是，Cinetry 支持多服务器一键切换。只需提前保存服务器配置，就能在多个数据源之间无缝切换，没有反复输入地址和账户的繁琐操作。

2. 全局化精准搜索

普通播放器只能单源搜索，而 Cinetry 能同时查询多个服务器的内容，实现全局化搜索结果。它支持标题、演员、导演、年份等多条件精准搜索，还能进行模糊匹配，哪怕记不清完整片名也能轻松找到。

搜索结果会按相关性智能排序，还能按电影、电视剧、音乐等类型一键过滤，不用在海量资源里翻找，想要的内容一键直达。

3. 极致播放体验

在播放体验上，Cinetry 内置高效播放器，支持多种音视频格式解码，还能完美适配 HDR、Dolby Vision 和 Dolby Atmos 等高规格内容，还原影音内容的原生质感。

4. 纯本地体验与隐私保护

值得一提的是，Cinetry 坚持纯本地体验，所有数据都存储在本地设备中，不向任何服务器上传个人信息，也不会主动扫描设备文件，仅加载手动导入的数据源，避免不必要的权限请求。

5. 权限管理与轻量化设计

此外，Cinetry 还设计了完善的权限管理功能，兼顾家庭使用需求。家长可以通过精细化设置，按内容类型、评级限制儿童访问，设置密码保护。

轻量化的设计让 Cinetry 拥有极速体验，启动快、播放流畅，哪怕配置多个服务器也不会卡顿。使用无需复杂操作，新手也能快速上手。对于影音爱好者来说，Cinetry 不仅是一款播放器，更是一站式的影音资源管理中心。

未来展望

据作者表示，除了电脑、手机、电视端外，目前 Web 端也在开发中。上线之后可以直接在 NAS 上部署，结合 NAS 的存储特点，对于 NAS 用户来说将会更加便捷，可以继续期待一下。
