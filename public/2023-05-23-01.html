PVE配置Docker之搭建自己的ChatGPT

杆子 2023-05-23 00:36  网络 234 字  1 分钟  编辑

Docker 安装过程可参考下面代码：

Ubuntu 安装好后，接下来就是安装 Docker 了。

安装 Docker

一键安装命令：

wget -qO- get.docker.com | bash

查看安装成功：

docker -v  # 查看 docker 版本

设置开机启动：

systemctl enable docker  # 设置开机自动启动

安装 Docker-compose

安装命令：

sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

设置权限：

sudo chmod +x /usr/local/bin/docker-compose

查看版本：

docker-compose --version  # 查看 docker-compose 版本

安装好 Docker 后，我们新建文件夹用于部署 ChatGPT：

sudo -i

mkdir -p /root/data/docker_data/chatgpt

cd /root/data/docker_data/chatgpt

创建 docker-compose 文件：

vim docker-compose.yml

在 docker-compose.yml 中添加 ChatGPT 安装命令（请根据实际使用的镜像修改）：

version: '3.8'
services:
  chatgpt-web:
    image: chenzhaoyu94/chatgpt-web:latest
    container_name: chatgpt-web
    ports:
      - 3002:3002
    environment:
      OPENAI_API_KEY: YOUR_API_KEY
      OPENAI_API_BASE_URL: https://api.openai.com
    restart: always

运行容器安装命令：

cd /root/data/docker_data/chatgpt    # 来到 docker-compose 文件所在的文件夹下

docker-compose up -d

到此 ChatGPT 安装完成。
