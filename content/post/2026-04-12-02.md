#### 安装 Claude Code {#claudecode}

在 Windows PowerShell 中执行以下命令进行安装：

```
curl -fsSL https://claude.ai/install.sh |bash

```

#### 安装 CC Switch {#ccswitch}

**项目地址：**  
https://github.com/farion1231/cc-switch

**下载软件：**  
https://github.com/farion1231/cc-switch/releases/tag/v3.12.3

下载并安装 `CC-Switch-v3.12.3-Windows.msi` 文件即可。

#### 配置 CC Switch {#ccswitch-1}

##### 1. 添加模型 {#1}

点击添加模型，界面如下：

![mnjopzy3.png](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/uP9BAA9ItAGRBhewjibGCSemricT1hEDLickKfSbYKvmfGvLibz5GQG8z0NeqYKNVjibOtVtvwf3scliayd94icibOAdF8027J5AQcMSGlB7wTbrauQ/640?wx_fmt=png&from=appmsg) 

例如，你可以添加 **Longcat** 的 `LongCat-Flash-Lite`，该模型每天提供 **5000万免费 Token**。

此外，**OpenRouter** 也有丰富的免费模型可供选择：  
https://openrouter.ai/models?fmt=cards&q=free

![mnjoqcr0.png](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/uP9BAA9ItAHOTmnMvxiaOLsKvFatY4iaaakZmKYQtdticbhZPzUK38ALoe3iaCYoPt1iapvNFjAboPVtO43fqQ8BcVXUD9F5bs1ggR1yA0fiazYlc/640?wx_fmt=png&from=appmsg)
