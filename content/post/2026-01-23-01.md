**原创：羊刀仙**

本期分享一个“自动签到”小项目，旨在解放双手、减少重复操作，将精力留给更重要的事情。简单来说，它的功能类似于青龙面板，但上手难度更低。

![封面图](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_jpg/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEG7p12PoYn95jRXyQ8Z282J3qbeA1uDCuO0ARhBupjE5qQx1BGNQOATA/0?wx_fmt=jpeg) 

#### 项目介绍 {#}

  * **项目名称**：qdtoday/qd（可直接在 GitHub 搜索）

  * **核心功能**：
    
      * ✅ **多平台支持**：内置 1000+ 网站模板，同时也支持自定义站点模板并上传。
      * ✅ **多账号管理**：同一平台可添加多个账号，集中管理无需来回切换。
      * ✅ **定时任务**：自定义执行时间，支持自动更新、自动清理日志等维护操作。
      * ✅ **多种通知**：支持 Bark、钉钉、企业微信等，执行结果可实时推送。
      * ✅ **图形化管理**：Web 界面配置，减少手动修改配置文件的麻烦。
      * ✅ **自动更新**：日常使用维护成本更低。

  * **⚠️ 使用提醒**：本项目仅用于个人学习与自动化实验，请遵守各平台用户协议与相关规定，勿用于违规用途或高频滥用。

#### 部署流程 {#-1}

本文以威联通（QNAP）NAS 为例，通过 Docker Compose 的方式进行部署。

部署代码如下：

```
services:
  qd:

# [补充内容] 指定镜像版本，latest 为最新版
    image: qdtoday/qd:latest
    container_name: qd
    restart: always
    ports:
      - 5678:80

# [补充内容] 根据需要挂载配置目录，持久化数据
    volumes:
      - ./config/config:/config

```

操作步骤：打开威联通的 Container Station，创建新的应用程序，将上述代码粘贴并应用即可。

#### 使用介绍 {#-2}

部署完毕后，在浏览器输入 `NAS_IP:5678` 即可访问服务。

![Web 访问界面](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGCT4DDzpaKWRp9wxuJsIna4UbExrpW2No7Rd8qdq1s5GB9cjQqDhTibA/640?wx_fmt=png) 

**1. 注册登录**

点击右上角的「登录」。我们先注册一个账户，这里的邮箱无需真实，只要格式符合要求即可。

![登录与注册](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGvwtDPDcKObO6ocUb2KLzMlicPNOyKDmiaKCRcyAtibRBdx42AYiaHqxyyQ/640?wx_fmt=png) 

**2. 订阅公共模板**

登录进来后，如下图所示，点击「公共模版」。

![公共模版入口](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGqHkyuQXgBkt0QQrwiaak4SaMGaZpOKiaqY0g4NE1P0BITKdtTCPwL3Qw/640?wx_fmt=png) 

等待自动更新完成后，所有模板便会罗列出来。顶部支持快捷搜索。找到你需要的站点，点击左侧的「订阅」。

![搜索并订阅模版](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGkbROnyaJRRNgTKpQAkqLQiazLghawGOfavnvqgRlShUjXiavlIR4c9tg/640?wx_fmt=png) 

以下图为例，演示具体操作。

![模版示例](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEG3FkWwnen5PdhWoicUFuCwAtDvOEHVUwudjIAeSMiarleowtol8D6bRjA/640?wx_fmt=png) 

点击「订阅」，然后将页面拉到底，点击「保存」。

![保存订阅](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGb1KtAjIvto0wbhs2j76Q5zMwzgYd4akv9SKNia1WnrtKRiaL4jAzpA1g/640?wx_fmt=png) 

在弹出的窗口内，补充好相关信息，设置好打卡频率，然后保存。

![设置任务参数](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGTDicEeY72797PMTmbmKpwntSVRicrfGHnicozAASrVtOysyp8KsHvSIibQ/640?wx_fmt=png) 

**3. 配置账号执行任务**

回到首页，便能看到刚才订阅的模版。

![首页模版列表](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGroc9qc8oiaxuJuvc9AbLOrsgJg3R7zjicmgF6TUvRfxoR0JCSPklyiclw/640?wx_fmt=png) 

接着在对应模版点击「新建」。

![新建任务](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGR1AdH17etwOeu2XuT9YlyID9Yexeg11yBib3fvjmanOTPDc1ZlZvlMg/640?wx_fmt=png) 

每个模版下都有备注说明，我们基本填写必填项目即可。演示案例中，只需要填写 **Domain**（域名）、**用户名**、**密码** 这三项。填写完毕后点击提交。

![填写账号信息](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGibZZW5eGiaia5rzSqDJqaLEdRzT32LLOFb89XwOjEd0L0bab9qYiaDlQug/640?wx_fmt=png) 

第一次测试时，访问域名可能失败（网络问题），可以看到运行结果。更换另一个站点测试，可以看到签到成功的提示。

在界面顶部，可以配置消息推送，例如将签到成功、失败等讯息推送到手机。

![配置推送通知](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/DD2l6bIOQMRSEDBiaqhm2kYlRUe1a0MEGwQHRwicHoKbWto4nVUHsgSPtnIksAx1xBkUDgNcnSIg4VZPljEOV0icw/640?wx_fmt=png)
