# 零服务器！把“网盘”直接跑在 Cloudflare 上，这个开源项目太狠了！

现在文件传输真的越来越“折磨人”：

👉 网盘限速 + 会员套路  
👉 微信/QQ各种限制  
👉 自建服务器？成本 + 运维直接劝退

有没有一种方案：

👉 不用服务器  
👉 成本接近 0  
👉 还能自己掌控数据

答案是：有。

👉 **EdgeStash**

但先别急着激动，这项目确实很强，但也有个现实门槛，我后面会说清楚。

---

## **⚡ 最大亮点：真的“零服务器”**

EdgeStash 最核心的一点就是：

👉 **完全不需要服务器**

它直接运行在：

✔ Cloudflare Workers（逻辑处理）  
✔ Cloudflare R2（文件存储）  
✔ Cloudflare KV（部分数据存储）

也就是说：

👉 不买 VPS  
👉 不用部署后端  
👉 不用维护环境

**一个 Cloudflare 账号，全搞定。**

---

## **☁️ 架构有多先进？**

传统网盘：

👉 服务器 + 数据库 + 硬盘

EdgeStash：

👉 Workers = 后端  
👉 R2 = 硬盘  
👉 KV = 数据库

一句话总结：  
**直接把“服务器架构”搬到了云边缘。**

---

## **📦 必须注意：R2 是核心依赖**

想用 EdgeStash，有个前提：

👉 **必须开通 Cloudflare R2**

你需要：

✔ 创建 Bucket  
✔ 绑定项目  
✔ 用来存储所有文件

没有 R2：

👉 项目无法运行

---

## **⚠️ 真实劝退点：需要信用卡**

重点来了👇

👉 开通 R2 时  
👉 **必须绑定信用卡（Visa / MasterCard）**

哪怕你：

✔ 用免费额度  
✔ 不打算付费

👉 也必须绑定

---

## **🤯 这就是很多人卡住的地方**

现实情况是：

❌ 没信用卡 → 用不了  
❌ 不想绑卡 → 放弃  
❌ 对支付敏感 → 犹豫

所以：

👉 技术门槛不高  
👉 **但“支付门槛”拦住了一部分人**

---

## **🔥 核心特性（这才是它真正强的地方）**

很多人以为它只是“文件中转工具”，  
其实完全不是，它已经接近一个完整网盘系统👇

---

### **👑 多用户系统（很少见）**

👉 管理员

* 最高权限
* 可管理用户 / 分享 / 数据统计

👉 授权用户

* 管理员创建（邮箱+密码）
* 只能浏览文件首页

👉 普通用户

* 通过密码访问分享文件
* 无密码文件可直接下载

**一句话：权限体系非常清晰，适合多人使用。**

---

### **⏳ 灵活的文件分享**

👉 支持设置访问密码  
👉 支持有效期：

* 1小时 / 1天 / 1个月 / 永久

👉 自动统计：  
✔ 浏览次数  
✔ 下载次数

**不仅能分享，还能“看数据”。**

---

### **🧸 在线文件预览（非常实用）**

不用下载，直接浏览器打开：

📄 文档：

* Word / PDF / Markdown / TXT

🖼 图片：

* JPG / PNG / GIF / WebP / SVG

📊 数据：

* JSON 自动格式化

🎵 音视频：

* MP4 / WebM / MP3 / WAV

**这点直接秒很多传统网盘体验。**

---

### **📁 强大的文件管理**

👉 文件 / 文件夹管理

* 创建 / 重命名 / 移动 / 删除

👉 上传体验：  
✔ 拖拽上传  
✔ 多文件上传

👉 面包屑导航

* 多层目录随意切换

---

### **🔧 完整管理后台**

👉 数据统计：

* 总分享数
* 总浏览量
* 总下载量

👉 用户管理：

* 添加 / 删除授权用户

👉 分享管理：

* 查看 / 删除所有分享链接

---

### **💡 现代化界面（不像开源项目）**

👉 紫蓝渐变设计（颜值在线）  
👉 响应式布局（手机也很好用）  
👉 动画流畅，交互舒服

---

### **🙈 安全与隐私**

👉 所有数据：

* 存在你的 R2 + KV

👉 登录系统：

* 基于 JWT（安全会话）

👉 不经过第三方服务器

**一句话：你才是数据的掌控者。**

---

## **🧠 它适合谁？**

✔ 有信用卡 / 虚拟卡的人  
✔ 经常传文件的人  
✔ 想低成本自建工具的人  
✔ 做资源站 / 下载站的人  
✔ 技术玩家 / 折腾党

不适合：

❌ 完全小白  
❌ 不想绑定支付方式

---

## **📌 总结（说人话版）**

EdgeStash 本质是：

👉 一个“跑在 Cloudflare 上的网盘系统”

优点很明显：

✔ 零服务器  
✔ 成本极低  
✔ 功能完整  
✔ 体验优秀

但也有现实限制：

👉 **必须绑定信用卡开通 R2**
