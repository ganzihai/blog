## 1. Google Gemini API

### 免费额度

- **Gemini 2.0 Flash / Flash-Lite**：15 RPM（每分钟请求数），150 万 Token/天
- **Gemini 2.0 Pro**：2 RPM，5 万 Token/天

### 使用方式

1. 访问 [Google AI Studio](https://aistudio.google.com/)
2. 点击 "Get API Key"，创建 API Key
3. 在代码中使用，示例：

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")
model = genai.GenerativeModel('gemini-2.0-flash')
response = model.generate_content("你好，请介绍一下自己")
print(response.text)
```

### 注意事项

- 免费版本的数据可能会被 Google 用于改进模型
- 有速率限制，不适合高并发场景
- 输出内容可能包含 Gemini 的水印

## 2. OpenAI API

### 免费额度

- **新注册用户**：赠送 $5 额度（有效期 3 个月）
- **gpt-4o-mini**：$0.15/百万输入 Token，$0.6/百万输出 Token

### 使用方式

1. 注册 [OpenAI 账号](https://platform.openai.com/)
2. 在 API Keys 页面创建新的 Key
3. 使用示例：

```python
from openai import OpenAI

client = OpenAI(api_key="YOUR_API_KEY")
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)
```

### 注意事项

- 免费额度用完后需要充值
- 中国大陆访问可能需要代理
- API 调用有速率限制

## 3. 百度文心一言 API

### 免费额度

- **ERNIE-Bot**：每日 10 万 Token 免费
- **ERNIE-Speed**：每日 50 万 Token 免费

### 使用方式

1. 注册 [百度智能云账号](https://cloud.baidu.com/)
2. 开通千帆大模型平台
3. 创建应用获取 API Key 和 Secret Key

```python
import requests

def get_access_token(api_key, secret_key):
    url = f"https://aip.baidubce.com/oauth/2.0/token?grant_type=client_credentials&client_id={api_key}&client_secret={secret_key}"
    response = requests.post(url)
    return response.json().get("access_token")

def chat(access_token, prompt):
    url = f"https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/eb-instant?access_token={access_token}"
    payload = {
        "messages": [{"role": "user", "content": prompt}]
    }
    response = requests.post(url, json=payload)
    return response.json().get("result")
```

### 注意事项

- 需要实名认证
- 有调用频率限制
- 部分高级模型需要付费

## 7. MiniMax

### 免费额度

- **abab6.5**：新用户赠送 1000 万 Token
- **abab5.5**：每日有一定免费额度

### 使用方式

1. 注册 [MiniMax 账号](https://api.minimax.chat/)
2. 获取 API Key
3. 使用示例：

```python
import requests

def chat(prompt):
    url = "https://api.minimax.chat/v1/text/chatcompletion_pro"
    headers = {"Authorization": "Bearer YOUR_API_KEY"}
    payload = {
        "model": "abab6.5",
        "messages": [{"role": "user", "content": prompt}]
    }
    response = requests.post(url, headers=headers, json=payload)
    return response.json().get("reply")
```

### 注意事项

- 需要实名认证
- 免费额度有限
- 适合中文场景
