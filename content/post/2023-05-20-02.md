## 内容

### 安装&简单配置 PVE (Proxmox VE)

#### 1. PVE (Proxmox VE) 下载地址

[Download Proxmox software, documentation, agreements](https://www.proxmox.com/en/downloads)

![PVE 下载页面](https://img.zznuo.com/i/2023/06/10/vhadnn-1.webp)

#### 2. 制作安装 U 盘

找个 U 盘制作 PVE 安装 U 盘，制作工具很多自行选择，这里用 Rufus，方法如下

![制作 U 盘](https://img.zznuo.com/i/2023/06/10/vhlg9f-1.webp)

#### 3. 安装 PVE 系统

制作好的 U 盘插到电脑 U 盘启动进行安装

![安装界面](https://img.zznuo.com/i/2023/06/10/vhow25-1.webp)

键盘方向键移动到 Install Proxmox VE 进行安装

![选择安装](https://img.zznuo.com/i/2023/06/10/vhs8ei-1.webp)

点击同意

![同意协议](https://img.zznuo.com/i/2023/06/10/vhtojg-1.webp)

选择安装硬盘，下一步

![选择硬盘](https://img.zznuo.com/i/2023/06/10/vhuwse-1.webp)

默认或者按需，下一步

![磁盘选项](https://img.zznuo.com/i/2023/06/10/vhwf1y-1.webp)

设置密码，邮件要写，下一步

![设置密码](https://img.zznuo.com/i/2023/06/10/vj3nap-1.webp)

按需设置，这里的设置的 IP 就是登录 PVE 要使用到的 IP，下一步

![网络设置](https://img.zznuo.com/i/2023/06/10/vjeirj-1.webp)

点 Install 进行安装

![开始安装](https://img.zznuo.com/i/2023/06/10/vjggsy-1.webp)

出现这个画面说明安装成功，记住链接

#### 4. 登陆 PVE 界面

`https://192.169.10.xx:8006` # 注意！地址前面一定要加 https

![登录页面](https://img.zznuo.com/i/2023/06/10/vji421-1.webp)

输入链接出现如上画面，点击高级，点击链接进入 PVE web 画面

![安全警告](https://img.zznuo.com/i/2023/06/10/vjk57g-1.webp)

输入用户名：root，密码，语言选择能看懂的，登录

![登录成功](https://img.zznuo.com/i/2023/06/10/vjmcya-1.webp)

点掉订阅提示

#### 5. 建议

上述设置完后 PVE 就能使用了，不熟悉的情况下不建议乱修改设置，稳定最重要！
