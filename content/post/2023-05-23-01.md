Docker安装过程可参考下面代码：  
Ubuntu安装好后，接下就是安装Docker了，  
>>>安装 Docker  
一键安装命令：

<pre class="line-numbers">`
wget -qO- get.docker.com | bash
`</pre>

查看安装成功：

<pre class="line-numbers">`
docker -v  #查看 docker 版本
`</pre>

设置开机启动：

<pre class="line-numbers">`
systemctl enable docker # 设置开机自动启动
`</pre>

>>>安装 Docker-compose：  
安装命令：

<pre class="line-numbers">`
sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
`</pre>

设置权限：

<pre class="line-numbers">`
sudo chmod +x /usr/local/bin/docker-compose
`</pre>

查看版本：

<pre class="line-numbers">`
docker-compose --version  #查看 docker-compose 版本
`</pre>

安装好Docker后，我们新建文件夹用于部署ChatGPT：

<pre class="line-numbers">`
sudo -i

mkdir -p /root/data/docker_data/chatgpt

cd /root/data/docker_data/chatgpt
`</pre>

创建docker-compose文件：

<pre class="line-numbers">`
vim docker-compose.yml
`</pre>

在docker-compose.yml中添加Portainer安装命令：

<pre class="line-numbers">`
version: &#039;3.8&#039;
services:
  portainer:
    image: cr.portainer.io/portainer/portainer-ce:2.11.0
    container_name: portainer
    restart: always
    ports:
      - 8000:8000
      - 9000:9000
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - /root/data/docker_data/portainer/data:/data
`</pre>

运行容器安装命令：

<pre class="line-numbers">`
cd /root/data/docker_data/chatgpt    # 来到 dockercompose 文件所在的文件夹下

docker-compose up -d 
`</pre>

到此ChatGPT安装完成。
