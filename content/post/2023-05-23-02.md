Docker安装过程可参考下面代码：  
Ubuntu安装好后，接下就是安装Docker了，  
>>>安装 Docker  
一键安装命令：

<pre class="line-numbers">`
wget -qO- get.docker.com | bash
`</pre>

查看安装成功：

<pre class="line-numbers">`
docker -v  #查看 docker 版本
`</pre>

设置开机启动：

<pre class="line-numbers">`
systemctl enable docker # 设置开机自动启动
`</pre>

>>>安装 Docker-compose：  
安装命令：

<pre class="line-numbers">`
sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
`</pre>

设置权限：

<pre class="line-numbers">`
sudo chmod +x /usr/local/bin/docker-compose
`</pre>

查看版本：

<pre class="line-numbers">`
docker-compose --version  #查看 docker-compose 版本
`</pre>

安装好Docker后，我们新建文件夹用于部署FRP：

<pre class="line-numbers">`
sudo -i

mkdir -p /root/data/docker_data/frpc

cd /root/data/docker_data/frpc
`</pre>

创建docker-compose文件：

<pre class="line-numbers">`
vim docker-compose.yml
`</pre>

在docker-compose.yml中添加Portainer安装命令：

<pre class="line-numbers">`
version: &#039;3&#039;
services:
    frpc:
        restart: always
        network_mode: host
        volumes:
            - &#039;./frpc.ini:/etc/frp/frpc.ini&#039;
        container_name: frpc
        image: snowdreamtech/frpc
`</pre>

运行容器安装命令：

<pre class="line-numbers">`
cd /root/data/docker_data/frpc    # 来到 dockercompose 文件所在的文件夹下

docker-compose up -d 
`</pre>

到此FRPC安装完成。
