Docker 安装过程可参考下面代码：

Ubuntu 安装好后，接下来就是安装 Docker 了。

### 安装 Docker

一键安装命令：

```bash
wget -qO- get.docker.com | bash
```

查看安装成功：

```bash
docker -v  # 查看 docker 版本
```

设置开机启动：

```bash
systemctl enable docker  # 设置开机自动启动
```

### 安装 Docker-compose

安装命令：

```bash
sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
```

设置权限：

```bash
sudo chmod +x /usr/local/bin/docker-compose
```

查看版本：

```bash
docker-compose --version  # 查看 docker-compose 版本
```

安装好 Docker 后，我们新建文件夹用于部署 EasyImage：

```bash
sudo -i

mkdir -p /root/data/docker_data/easyimage

cd /root/data/docker_data/easyimage
```

创建 docker-compose 文件：

```bash
vim docker-compose.yml
```

在 docker-compose.yml 中添加 EasyImage 安装命令：

```yaml
version: '3.3'
services:
  easyimage:
    image: ddsderek/easyimage:latest
    container_name: easyimage
    ports:
      - '8088:80'
    environment:
      - TZ=Asia/Shanghai
      - PUID=1000
      - PGID=1000
    volumes:
      - '/root/data/docker_data/easyimage/config:/app/web/config'
      - '/root/data/docker_data/easyimage/i:/app/web/i'
    restart: unless-stopped
```

运行容器安装命令：

```bash
cd /root/data/docker_data/easyimage    # 来到 docker-compose 文件所在的文件夹下

docker-compose up -d
```

到此 EasyImage 安装完成。
