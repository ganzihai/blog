## 三、详细开启教程（Windows & Mac） {#windowsmac}

此过程涉及修改系统文件，请严格按照顺序操作。

### 第一步：彻底关闭 Chrome {#chrome}

由 90% 的失败源于此——很多人改了文件没生效，是因为 Chrome 只是关闭了窗口，后台进程还在运行，导致修改被覆盖。

  - **Windows 用户**：打开任务管理器（Ctrl+Shift+Esc），找到所有 `Google Chrome` 进程，右键点击 **&#8220;结束任务&#8221;**，确保一个不留。
  - **Mac 用户**：在 Dock 栏对着 Chrome 图标右键 -> **退出**，或者打开&#8221;活动监视器&#8221;强制退出所有 Chrome 相关进程。

### 第二步：修改本地配置文件 {#-2}

找到 Chrome 的 `Local State` 文件，这是一个没有后缀名的 JSON 文件。

**文件路径**：

  - **Windows**: `C:\Users\你的用户名\AppData\Local\Google\Chrome\User Data\Local State`
  - **Mac**: `~/Library/Application Support/Google/Chrome/Local State`

![Local State文件位置](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/IwWVXbrA7gf1fESPYPKWP4f64gHc9Au41b8wwToibJ9d6ITN3h1ox88KyjQIxlFwCL6QLpYWRHuHzSl0fugSVvBA6PgyU5FNibiaA4wxXrL4eI/640?wx_fmt=png&from=appmsg)

#### 1. 修改 variations_country {#1variations_country}

  - 原代码地区可能是：`"cn"` 或其他地区
  - **改为**：`"us"`
  - 注意：保留前面的数字版本号，只改国家代码。

![修改variations_country](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/IwWVXbrA7gchrvDKV1XyU2p0mVSDBkFMsFUlzCWh6SeHwvARNB3Zgy6a7X5tjvrgqZ3NdqCxlyF05PvjC6p7v6iap3rSoia7H9Cw6mDTDTtRc/640?wx_fmt=png&from=appmsg)

#### 2. 修改 variations_country（第二处） {#2variations_country}

  - 原代码：`"variations_country":"cn"`
  - **改为**：`"variations_country":"us"`

![修改第二处variations_country](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/IwWVXbrA7gdpFrFq31EXQGp2Kwll5Uax2xxC5VqbUw6hFRic3BqN1dtqfKPxbLzrib1NGJXTOQ82C0zlhOxhLCj6TpTjdVOxDy8H8ACiaib5028/640?wx_fmt=png&from=appmsg)

#### 3. 修改/添加 is_glic_eligible {#3is_glic_eligible}

  - 搜索 `is_glic_eligible`
  - 如果找到了，把值改为 `true`（可能有好几处，都修改一下）
  - 如果找不到：请在文件靠前的位置（例如第一个大括号 `{` 之后）手动添加一行：`"is_glic_eligible": true,`

![修改is_glic_eligible](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/IwWVXbrA7gfiaHHzQeCz9DlBsH4PjlT33sfLniaMdtQLibUykqSDx1htY29gUpHTIrPoAhOADuiaKTpXcvBNu8yON4dgOGWPbBZTHUjicunvh56U/640?wx_fmt=png&from=appmsg)

**全部修改完后，保存文件并关闭。**

### 第三步：添加启动参数（增加成功率的&#8221;保险&#8221;） {#-3}

很多教程忽略了这一步，导致功能时有时无。

#### Windows 用户 {#windows}

  1. 右键点击桌面的 Chrome 快捷方式 -> **属性**
  2. 找到 **&#8220;目标&#8221;** 一栏，在原有路径的最后面，**先打一个空格**，然后粘贴以下代码：

```
--enable-features=OptimizationGuideModelDownloading,OptimizationHints
```

<ol start="3">
  <li>
    点击确定。以后<strong>必须通过这个快捷方式启动 Chrome</strong>。
  </li>
</ol>

#### Mac 用户 {#mac}

打开终端（Terminal），输入以下命令后回车（这是最稳妥的方式）：

```
open -a "Google Chrome" --args --enable-features=OptimizationGuideModelDownloading,OptimizationHints
```

### 第四步：系统与浏览器环境&#8221;伪装&#8221; {#-4}

#### Windows 设置 {#windows-1}

打开&#8221;设置&#8221; -> &#8220;时间和语言&#8221; -> &#8220;区域&#8221;，将&#8221;国家或地区&#8221;改为 **美国**。

#### Mac 设置 {#mac-1}

打开&#8221;系统设置&#8221; -> &#8220;通用&#8221; -> &#8220;语言与地区&#8221;，将&#8221;地区&#8221;改为 **美国**。

![系统地区设置](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/IwWVXbrA7ge6cZniakVeiaPh5GVmhwcCO4sQdyxicUibOe8IkXjVFp94BuVfRx6pya4sdz7NxeObKVVjnhn4uxibFBTA1EJVjgd9ibTSXhUibbo3iak/640?wx_fmt=png&from=appmsg)

#### Chrome 内部语言设置（关键） {#chrome-1}

打开 Chrome 设置 (`chrome://settings/languages`)，将 **English (United States)** 添加并置顶，勾选&#8221;以这种语言显示 Google Chrome&#8221;。**这一步非常重要，中文界面极大概率无法触发 Gemini。**

![Chrome语言设置](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/mmbiz_png/IwWVXbrA7geNcc1sdVqG4OKNr25D0LiclW2puA3cq2mF8Ygx1bw1B9m5hJazGf2FKiaRNan1lH1ribqOYMJoOKJUJoC5UdOsUG1ycPUQIOfZbY/640?wx_fmt=png&from=appmsg)

### 第五步：开启 Flags 开关 {#flags}

重启浏览器后，在地址栏输入 `chrome://flags`，搜索 **&#8220;glic&#8221;**，将出现的以下两项从 Default 改为 **Enabled**：

  - `#glic` (Gemini Live in Chrome)
  - `#glic-side-panel`

最后点击右下角的 **Relaunch** 再次重启。

![Flags开关设置](https://wsrv.nl/?output=webp&url=https://mmbiz.qpic.cn/sz_mmbiz_png/IwWVXbrA7gfKvibicQq3f6woLx3xARqXIicgusZ2DnDkg5e0tIQJw79lkeQeoBrwcNdE9sG3Jh84nTJcAU3PQ8WdwoaDqRrzAuaDvFzndGZnV8/640?wx_fmt=png&from=appmsg)
