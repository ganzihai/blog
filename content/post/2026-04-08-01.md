### 背景介绍 {#}

想用 OpenClaw、想在自己工具里集成 API，但 Token 太贵了？不过，各大商家不是都提供了免费的网页版吗？比如 doubao、ChatGPT，网页版是不限量还免费的！

所以这次介绍的工具，就是将网页版的 AI 转成了兼容 OpenAI 协议的 API。以前也有 web2api、chat2apt，不过都不更新了。而这次的 **WebAI2API** 非常好用。

### 部署教程 {#-2}

> **项目链接**：https://github.com/foxhui/WebAI2API

#### 方式一：手动部署 {#-3}

**1. 克隆项目**

```

git clone https://github.com/foxhui/WebAI2API.git

```

**2. 调整配置文件**

程序初次运行会从 `config.example.yaml` 复制配置文件到 `data/config.yaml`。建议先配置好 `server` 等相关参数。

**3. 安装与配置**

```

# 1. 安装 NPM 依赖

npm install

```

**4. 启动服务**

```

# 标准启动

npm start

```

#### 方式二：Docker 部署 {#docker}

**安全提醒**：

  - Docker 镜像默认开启虚拟显示器 和 VNC 服务。
  - 可通过 WebUI 的虚拟显示器板块连接。
  - **注意**：WebUI 传输过程未加密，公网环境请使用 SSH 隧道或 HTTPS。

**Docker CLI 启动**

```

docker run -d --name webai-2api \
  -p 5000:5000 \
  -v ./data:/app/data \
  foxhui/webai-2api:latest

```

**Docker Compose 启动**

```

docker-compose up -d

```
