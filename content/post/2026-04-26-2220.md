## 项目速览

**LinkSwift** 是一款基于 JavaScript 的浏览器用户脚本（油猴脚本），核心功能就一件事：**从各大网盘获取文件的真实下载直链**，绕过网盘客户端的限速和限制，用第三方下载工具满速下载。

![](https://wsrv.nl/?output=webp&url=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FdruA5NpNJekheXDpA0xv4uRjLuRuiaeIAxleYOxQRSSQDrsZkUJggbYKsgAEwbJmI6RDANQWLXhjap45mArGs5krFFaiaTuoUiay1SfVj53EoM%2F640%3Fwx_fmt%3Dpng)

这个项目基于早期知名的「网盘直链下载助手」修改而来，但经过了大量重构和功能扩展，支持范围远超原版。

## 怎么用？

### 安装（三步上手）

1**安装脚本管理器**：Tampermonkey（推荐）或 Violentmonkey

2**安装 LinkSwift 脚本**：前往 Greasy Fork 搜索 "LinkSwift"，点击安装

3**打开网盘页面**：进入任意支持的网盘分享页，勾选要下载的文件

安装完成后，网盘页面会自动出现「下载助手」按钮。

### 下载方式一览

| 方式                      | 适用场景               |
| ----------------------- | ------------------ |
| **IDM 推送**              | Windows 用户首选，满速多线程 |
| **Aria2 / RPC**         | NAS、服务器、路由器远程下载    |
| **比特彗星**                | BT 下载场景            |
| **AB Download Manager** | 开源下载管理工具           |
| **API 下载**              | 直接获取直链，浏览器下载或复制    |
| **cURL / Wget**         | 命令行用户，复制即用         |

所有方式都直接从浏览器弹出界面操作，无需额外软件配置（推送到 Aria2 等需自行部署对应服务）。

## 核心功能

### 1\. 多线程增强下载

内置「增强下载」模式，支持多块多线程并发，浏览器直接下载也能提速。

### 2\. 百度网盘文件夹下载

支持在百度网盘中选择整个文件夹批量获取下载链接，不用一个一个点。

### 3\. IDM 客户端无缝接管

链接可一键推送到 IDM（Internet Download Manager），自动嗅探并接管下载任务，32 线程满速拉取。

### 4\. 链接缓存

重复获取链接不必重新请求，本地缓存直达，节省时间。

### 5\. 界面主题自定义

支持更换 7 大网盘的界面主题颜色，内置深色模式，适配 Dark Reader。

### 6\. 多配置切换

支持添加、删除、切换多套 Aria2 / RPC 配置，家里 NAS、公司服务器、路由器切换无忧。

### 7\. 隐私安全

•废弃了百度网盘 BDUSS Cookie 获取方式，改用更安全的 AccessToken

•不使用第三方中转服务器

•远程配置可自主管理

## 技术原理（简单版）

LinkSwift 是一个用户脚本，不修改网盘客户端，也不劫持流量。它的工作方式是：

1**注入到网盘页面** → 自动识别当前页面属于哪个网盘

2**解析文件信息** → 读取页面 DOM 和 API 响应，提取文件列表和特征参数

3**请求下载地址** → 调用网盘的内部接口，获取带签名的临时直链

4**推送到下载工具** → 根据用户选择，将直链发送给 IDM、Aria2 或直接复制

整个过程在浏览器本地完成，不需要任何桌面级权限。

---

## 注意事项

1**仅支持分享页**：如果文件在自己的网盘里，需要先生成分享链接

2**依赖网盘接口**：各网盘不定期更新反爬策略，直链获取可能间歇性失效，一般几天内修复

3**非官方工具**：使用本脚本下载的文件直链，由网盘服务端生成，脚本本身不参与文件传输

4**百度网盘限制**：未开通超级会员的账号，即便获取到直链，速度仍可能受限
